# Rudraksh - Architecting Digital Growth

Digital Marketing Specialist and Strategist focused on AI workflow automation, Skool community management, and scaling digital brands through optimized operations.

## Technologies

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Animation**: Framer Motion

## Deployment

The project includes configuration for SPA routing on:
- **Netlify**: `public/_redirects`
- **Vercel**: `vercel.json`

This ensures that reloading any page (e.g., project details) correctly redirects to `index.html` and prevents 404 errors.
